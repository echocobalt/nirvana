# nirvana

my simple shell

## Installation:

### On Linux/macOS:

```bash
git clone https://github.com/echocobalt/nirvana.git
cd nirvana/src
python3 main.py
```

### On Windows:
```powershell
git clone https://github.com/echocobalt/nirvana.git
cd nirvana\src
python main.py
```

## Shorthand:

**Add this to your `~/.zshrc` || `~/.bashrc`**

```zsh
echo "alias nirvana=cd location/to/cloned/repo/src/main.py" >> ~/.zshrc # or ~/.bashrc
```
